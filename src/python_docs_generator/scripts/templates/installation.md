---

### Installation

#### From PyPI

```
pip install {{ project["name"] }}
```

#### From source (as package)

```
pip install git+https://github.com/{{ organization_name }}/{{ project["name"] }}.git
```

#### From source (as repo)

```
git clone https://github.com/{{ organization_name }}/{{ project["name"] }}.git
cd {{ project["name"] }}
pip install .
```
